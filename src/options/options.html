Warnlisten Settings

Configure URL analysis and API connection

🛡️ Analysis Services

Checks URLs against Google's threat database (v4)

Checks for consumer complaints on reklamation.ch

Checks for entries on the Ktipp warning list

🔌 API Configuration

Enter the URL of your warning list REST API. Default is Google Safe Browsing v4.

Your API key is stored locally and never shared. Leave empty for heuristic-only mode.

🛡️ Whitelist

Enter domains to skip from analysis (one per line). All subdomains will also be skipped.

⚙️ Behavior

Automatically analyze URLs when navigating to new pages

Display a badge on the page when threats are detected
