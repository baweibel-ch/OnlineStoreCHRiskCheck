# Deutsch (de)

# Datenschutzerklärung (Privacy Policy) für "CH Warnlisten & Shop Risk-Check"

Diese Datenschutzerklärung informiert Sie darüber, welche Daten von der Browser-Erweiterung "CH Warnlisten & Shop Risk-Check" erfasst, wie sie verwendet und an wen sie weitergegeben werden. Diese Erklärung entspricht den Datenschutzrichtlinien des Chrome Web Stores.

## 1. Welche Daten werden erfasst und verarbeitet?
Die Erweiterung erfasst **keine personenbezogenen Daten** wie Namen, E-Mail-Adressen oder Passwörter.
Um ihre Kernfunktion – den Schutz vor Fake-Shops und riskanten Webseiten – zu erfüllen, verarbeitet die Erweiterung folgende Informationen:
- **Domains und URLs der besuchten Webseiten**: Wenn die Prüfung (automatisch oder manuell) aktiviert ist, liest die Erweiterung die URL oder Domain des aktuellen Tabs aus, um diese auf Risiken zu überprüfen.

## 2. Wie werden die Daten verwendet und an wen werden sie übermittelt?
Um die Webseiten zu analysieren, übermittelt die Erweiterung die extrahierten Domains oder URLs an externe Dienste. Es findet **kein Tracking** durch uns statt, und die Daten werden **nicht** auf unseren eigenen Servern gespeichert.

Die Erweiterung prüft Online-Shop-Domains/URLs gegen Warnlisten, Reklamationen, Trusted Shops & Admin.ch UID. 
Dabei wird bei den Anfragen an die Suchdienste `OnlineStoreCHRiskCheck` als Referer (Verweisquelle) mitgegeben.
Dazu sendet sie die **Domain** an folgende Dienste:
* **K-Tipp/Saldo (Warnliste):** www.ktipp.ch/service/warnlisten/detail/warnliste/internetshops
  * *Datenschutz:* https://www.ktipp.ch/ueber-uns/datenschutz
* **Reklamation.ch:** www.reklamation.ch
  * *Datenschutz:* https://reklamation.ch/datenschutz.php
* **Unternehmens-Identifikationsnummer (UID):** www.uid.admin.ch
  * *Datenschutz:* https://www.admin.ch/de/rechtliches
* **Trusted Shops:** www.trustedshops.ch
  * *Datenschutz:* https://www.trustedshops.ch/impressum/#data-protection

Zudem wird die **URL** (optional) an folgenden Dienst gesendet:
* **Google Safe Browsing:**
  * *Datenschutz:* https://developers.google.com/safe-browsing / https://policies.google.com/privacy

## 3. Nutzerkontrolle und Einstellungen
Jede dieser Prüfungen ist in den Einstellungen der Erweiterung **optional zu setzen**. 
Sie können wählen, ob die Prüfung **manuell (empfohlen)** auf Knopfdruck oder **automatisch** beim Surfen erfolgen soll. Diese Präferenzen lassen sich jederzeit in den Einstellungen anpassen.

## 4. Datenspeicherung und Aufbewahrung
Die erfassten Domains und URLs werden ausschließlich in Echtzeit zur Durchführung der Sicherheitsprüfung verwendet. Sie werden weder lokal dauerhaft gespeichert, noch an Server der Entwickler dieser Erweiterung übermittelt.
Die Erweiterung nutzt den lokalen Speicher (Local Storage) des Browsers lediglich, um Ihre Benutzereinstellungen (z.B. API-Schlüssel für Google Safe Browsing, Whitelist) lokal auf Ihrem Gerät zu speichern.

## 5. Weitergabe an Dritte
Die Daten (URLs/Domains) werden ausschließlich an die oben in Abschnitt 2 genannten Dienste zur Sicherheitsprüfung weitergegeben. Die Daten werden nicht an Dritte verkauft oder für Werbezwecke verwendet.

## 6. Kontakt
Da diese Erweiterung Open Source ist, können Sie den Quellcode jederzeit einsehen. Bei Fragen zum Datenschutz können Sie ein Issue im GitHub-Repository eröffnen: https://github.com/baweibel-ch/OnlineStoreCHRiskCheck

# Privacy Policy for "CH Warnlisten & Shop Risk-Check"

This privacy policy informs you about what data is collected by the browser extension "CH Warnlisten & Shop Risk-Check", how it is used, and to whom it is shared. This statement complies with the Chrome Web Store privacy policies.

## 1. What data is collected and processed?
The extension collects **no personal data** such as names, email addresses, or passwords.
To fulfill its core function – protecting against fake shops and risky websites – the extension processes the following information:
- **Domains and URLs of the visited websites**: If the check (automatic or manual) is activated, the extension reads the URL or domain of the current tab to check it for risks.

## 2. How is the data used and to whom is it transmitted?
To analyze the websites, the extension transmits the extracted domains or URLs to external services. There is **no tracking** by us, and the data is **not** stored on our own servers.

The extension checks online shop domains/URLs against warning lists, complaints, Trusted Shops & Admin.ch UID.
In doing so, `OnlineStoreCHRiskCheck` is provided as the Referer when querying the search services.
To do this, it sends the **domain** to the following services:
* **K-Tipp/Saldo (Warning List):** www.ktipp.ch/service/warnlisten/detail/warnliste/internetshops
  * *Privacy:* https://www.ktipp.ch/ueber-uns/datenschutz
* **Reklamation.ch:** www.reklamation.ch
  * *Privacy:* https://reklamation.ch/datenschutz.php
* **Business Identification Number (UID):** www.uid.admin.ch
  * *Privacy:* https://www.admin.ch/de/rechtliches
* **Trusted Shops:** www.trustedshops.ch
  * *Privacy:* https://www.trustedshops.ch/impressum/#data-protection

In addition, the **URL** (optional) is sent to the following service:
* **Google Safe Browsing:**
  * *Privacy:* https://developers.google.com/safe-browsing / https://policies.google.com/privacy

## 3. User control and settings
Each of these checks is **optional to set** in the extension's settings.
You can choose whether the check should be done **manually (recommended)** at the push of a button or **automatically** while browsing. These preferences can be adjusted at any time in the settings.

## 4. Data storage and retention
The collected domains and URLs are used exclusively in real time to carry out the security check. They are neither permanently stored locally nor transmitted to the servers of the developers of this extension.
The extension only uses the local storage of the browser to store your user settings (e.g., API key for Google Safe Browsing, whitelist) locally on your device.

## 5. Sharing with third parties
The data (URLs/domains) are only passed on to the services mentioned above in Section 2 for security checks. The data is not sold to third parties or used for advertising purposes.

## 6. Contact
Since this extension is open source, you can view the source code at any time. If you have any questions about privacy, you can open an issue in the GitHub repository: https://github.com/baweibel-ch/OnlineStoreCHRiskCheck
