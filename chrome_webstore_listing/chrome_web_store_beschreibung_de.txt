Sicher einkaufen im Internet – Ihr Schutz vor Fake-Shops und Betrug

"CH Warnlisten & Shop Risk-Check" ist eine Browser-Erweiterung, die Webseiten und Online-Shops automatisch oder manuell auf Sicherheitsrisiken analysiert.

Prüft ob die Domain der aktuellen URL auf der Schweizer Warnliste von K-Tipp/Saldo oder Reklamation gelistet ist und und ob sie im UID-Register@BFS enthalten ist. Es wird ebenfalls auf Trustedshops nach Bewertungen gesucht. Optional werden Google Safe Browsing oder heuristische Prüfungen für die Url durchgeführt.

Es werden nur Internet-Adressen (https:// oder http://) analysiert, localhost oder IP (http://127.0.0.1 etc.) auch nicht.

Die Prüfung kann automatisch oder manuell eingestellt werden.

Automatische oder manuelle Überprüfung via:
- www.ktipp.ch/service/warnlisten/detail/warnliste/internetshops
- www.reklamation.ch
- www.uid.admin.ch
- - (Nicht unterstützt für Firefox-Android)
- www.trustedshops.ch
- Google Safe Browsing

Reduzierte Aktivität:
- Jede Domain wird nur 1x geprüft, egal ob Sie im gleichen Tab auf der Seite weiterklicken oder dieselbe Domain in anderen Tabs geöffnet haben.
- Wenn Google-Safe-Browsing aktiviert ist, wird jedoch jede neue URL erneut auf Sicherheitsrisiken geprüft.
- Im manuellen Modus, wenn die Prüfung ausgelöst wurde, werden für jede weitere Seite derselben Domain die Warnungen für die Domain ohne weitere Prüfung angezeigt und die URL via Google-Safe-Browsing getestet (falls aktiviert).

Da ich kein offizieller Entwickler einer der Web-Services bin, kann die Funktionalität eingeschränkt sein wenn sich die Services ändern.

Nach der Installation: 'An Symbolleiste anpinnen/anheften' - ansonsten macht es wenig Sinn. Dies kann jedoch nicht automatisiert werden.

Hauptfunktionen:
- 🛒 **K-Tipp/Saldo Warnliste:** Überprüft automatisch, ob eine Domain auf der offiziellen K-Tipp/Saldo Warnliste steht.
- - Falls ein Eintrag gefunden wird, wird dieser angezeigt. Der Link öffnet die Warnlisten-Seite, die Domain muss danach jedoch selbst eingetragen werden.
- 🛡️ **reklamation.ch Abgleich:** Sucht nach bestehenden Konsumentenbeschwerden, bevor Sie einkaufen.
- ✅ **Trusted Shops Zertifizierung:** Prüft automatisch, ob der Shop ein gültiges Trusted Shops Gütesiegel besitzt.
- 🏛️ **Admin.ch UID Prüfung:** Gleicht die Domain mit dem offiziellen Unternehmensregister des Bundes ab. Shops ohne gültige Schweizer UID (Unternehmens-Identifikationsnummer) werden als Warnung angezeigt.
- - (Nicht unterstützt für Firefox-Android)
- 🔒 **Google Safe Browsing (Optional):** Integration der Safe Browsing API zur Erkennung von Phishing und Malware.
- - Für diese Funktionalität müssen Sie einen API-Key in den Einstellungen hinterlegen. Ein Anleitungs-Link zum erstellen einer solchen ist dort angezeigt
- ⚙️ **Heuristische Prüfungen:** Lokale Erkennung von verdächtigen TLDs, unverschlüsselten HTTP-Verbindungen und URL-Verschleierung – ganz ohne API-Key.
- 🚦 **Sichtbare Warnungen:** Ein Farbsystem im Icon (Badge) und direkte Warnungen auf gefährlichen Seiten informieren Sie sofort.
- - Icon-Badges in der Symbolleiste des Browsers zeigen den aktuellen Status auf einen Blick:
- - – Sicherheitsbedrohung erkannt (rot, „❗“).
- - – Warnung / Beschwerden gefunden (orange, „❗“).
- - – Keine Auffälligkeiten gefunden (grün, „✅“).
- - – Domain auf Whitelist (weiss, „✔“).
- - – Noch keine Prüfung durchgeführt (weiss, „?“).
- - – Prüfung übersprungen, z. B. localhost/IP/nicht-Http(s) (grau, „-“).
- 🛡️ **Datenschutz an erster Stelle:**
- - Keine Nutzerverfolgung (Tracking), keine Speicherung von Browserverläufen auf unseren Servern.
- - 'OnlineStoreCHRiskCheck' wird den Prüf-Seiten als 'Referer' angegeben (HTTP-Header, der angibt, von welcher Webseite eine Anfrage kommt / die vorherige Seite, die den Nutzer zur aktuellen Seite weitergeleitet hat) wird

So funktioniert's:
- Wählen Sie zwischen automatischem Scannen beim Surfen oder manuellem Scannen auf Knopfdruck. Fügen Sie vertrauenswürdige Seiten bei Bedarf einer Whitelist hinzu.
- Eine Domain wird nur 1x geprüft pro Sitzung im aktuellen- und allen Tabs (für K-Tipp/Saldo Warnliste, Reklamation.ch, Trusted Shops, uid.admin.ch).
- Die Safe Browsing wird pro Url 1x ausgeführt.
- Ausser wenn die Prüfung explizit erneut ausgeführt wird.
