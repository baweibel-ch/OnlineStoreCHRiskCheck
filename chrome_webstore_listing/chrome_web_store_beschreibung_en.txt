Shop safely on the Internet – Your protection against fake shops and fraud

"CH Warnlisten & Shop Risk-Check" is a browser extension that automatically or manually analyzes websites and online shops for security risks.

It compares the current URL with the Swiss warning list from K-Tipp/Saldo (ktipp.ch/service/warnlisten/detail/warnliste/internetshops), reklamation.ch and uid.admin.ch. Optional Google-Safe-Browsing or heuristic checks.

The check can be set to automatic or manual.

Automatic or manuel verification via:
- www.ktipp.ch/service/warnlisten/detail/warnliste/internetshops
- www.reklamation.ch
- www.uid.admin.ch
- www.trustedshops.ch
- Google Safe Browsing (safebrowsing.googleapis.com)

Since I am not an official developer of the Ktipp/Saldo warning list or Reklamation.ch, and no API is found, the check is done via web requests. If the websites change, the functionality may be restricted afterwards.

After installation: 'Pin to toolbar' - otherwise it makes little sense. This cannot be automated, however.

Main features:
- 🛒 **K-Tipp/Saldo Warning List:** Automatically checks whether a domain is on the official K-Tipp/Saldo warning list.
- - If an entry is found, it will be displayed. The link opens the warning list page, but the domain must then be entered manually.
- 🛡️ **reklamation.ch Comparison:** Searches for existing consumer complaints before you shop.
- ✅ **Trusted Shops Certification:** Automatically checks whether the shop has a valid Trusted Shops seal of approval.
- 🏛️ **Admin.ch UID Check:** Compares the domain with the official federal business register. Shops without a valid Swiss UID (Business Identification Number) are displayed as a warning.
- 🔒 **Google Safe Browsing (Optional):** Integration of the Safe Browsing API to detect phishing and malware.
- - For this functionality, you need to provide an API key in the settings. A link to instructions on how to create one is displayed there.
- ⚙️ **Heuristic Checks:** Local detection of suspicious TLDs, unencrypted HTTP connections, and URL obfuscation - entirely without an API key.
- 🚦 **Visible Warnings:** A color system in the icon (badge) and direct warnings on dangerous sites inform you immediately.
- 🛡️ **Privacy First:** No user tracking, no storing of browsing history on our servers.


How it works:
- Choose between automatic scanning while surfing or manual scanning at the push of a button. Add trusted sites to a whitelist if necessary.
- A domain is checked only once per session in the current and all tabs (for K-Tipp/Saldo warning list, Reklamation.ch, Trusted Shops, uid.admin.ch).
- Safe Browsing is executed once per URL.
- Unless the check is explicitly run again.
